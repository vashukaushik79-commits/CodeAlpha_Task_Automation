# 📁 Task Automation with Python Scripts

An automation script built in Python to simplify file management. This project scans a source directory and automatically transfers all `.jpg` images into a designated target folder. Developed during my internship at **CodeAlpha**.

## 🚀 Features
* **Auto-Directory Creation:** Automatically creates source and destination folders if they don't exist.
* **Smart Filtering:** Accurately filters files using specific extensions (`.jpg`).
* **Safe Operations:** Uses `shutil.copy` to safely replicate files without deleting original data.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Modules:** `os`, `shutil`
